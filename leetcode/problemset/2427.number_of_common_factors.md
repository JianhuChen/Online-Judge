# 2427. 公因子的数目

[打开力扣](https://leetcode.cn/problems/number-of-common-factors) [我的代码](2427.number_of_common_factors.py)

给你两个正整数 <code>a</code> 和 <code>b</code> ，返回 <code>a</code> 和 <code>b</code> 的 <strong>公</strong> 因子的数目。

如果 <code>x</code> 可以同时整除 <code>a</code> 和 <code>b</code> ，则认为 <code>x</code> 是 <code>a</code> 和 <code>b</code> 的一个 <strong>公因子</strong> 。



<strong>示例 1：</strong>

<pre><strong>输入：</strong>a = 12, b = 6
<strong>输出：</strong>4
<strong>解释：</strong>12 和 6 的公因子是 1、2、3、6 。
</pre>

<strong>示例 2：</strong>

<pre><strong>输入：</strong>a = 25, b = 30
<strong>输出：</strong>2
<strong>解释：</strong>25 和 30 的公因子是 1、5 。</pre>



<strong>提示：</strong>

<ul>
	<li><code>1 <= a, b <= 1000</code></li>
</ul>
